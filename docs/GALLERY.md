# 🏆 GALERIA DE EXCELÊNCIA (INSPIRAÇÃO SUPREMA)

Para sair da caixa, você deve estudar os mestres. Aqui estão os padrões de design que elevaram sites comuns à categoria de arte.

## 1. O Scroll Narrativo (The Storyteller)
- **Referência**: `aventuradentalarts.com`
- **Padrão**: O site não tem scroll vertical tradicional. A tela trava e o conteúdo desliza horizontalmente, revelando seções como um filme.
- **DNA Argus**: Use `pinned-section.js` + `horizontal-scroll.js`.

## 2. A Tipografia como Arquitetura (The Architect)
- **Referência**: `igloo.inc`
- **Padrão**: O texto é tão grande que ele se torna a borda do site. Ele sangra para fora da tela e interage com o cursor.
- **DNA Argus**: Use `DepthText.tsx` e `mix-blend-mode: difference`.

## 3. O Objeto Físico Digital (The Tactile)
- **Referência**: Interfaces de sintetizadores (Teenage Engineering).
- **Padrão**: Cada botão parece que pode ser apertado. As sombras mudam de ângulo conforme o mouse se move.
- **DNA Argus**: Use `Tactile Skeuomorphism` + `spotlight.js`.

## 4. O Caos Controlado (The Rebel)
- **Referência**: Sites de agências vencedoras do SOTD.
- **Padrão**: Elementos sobrepostos, grids que mudam de forma, tipografia distorcida.
- **DNA Argus**: Use `GlitchText.tsx` e `InterlockingCards.tsx`.
