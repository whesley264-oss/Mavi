# ⚡ BIBLIOTECA DE PROMPTS ARGUS

Use estes prompts mestres para tarefas específicas. Eles foram desenhados para extrair o máximo da Skill Argus.

---

## 1. Prompt de Criação (Hero/Section)
> *"Argus, crie uma seção de [NOME DA SEÇÃO] no estilo [ESTILO ESCOLHIDO]. Siga a arquitetura definida em `docs/ARQUITETURA.md` e garanta que o código seja blindado seguindo `docs/SEGURANCA.md`. Use tipografia fluida e micro-interações magnéticas."*

## 2. Prompt de Auditoria Red Team
> *"Argus, assuma a persona Red Team agora. Ataque este arquivo: [CAMINHO DO ARQUIVO]. Identifique todas as 50 vulnerabilidades de `docs/SEGURANCA.md` que podem ser exploradas e apresente o patch de correção imediata."*

## 3. Prompt de Documentação
> *"Argus, documente este projeto seguindo o rigor de `docs/DOCUMENTACAO.md`. Crie uma visão geral, arquitetura detalhada e guia de manutenção para profissionais seniores."*

## 4. Prompt de Refatoração de Design
> *"Argus, este componente está genérico. Aplique o Atlas de Estilos de `docs/ESTILOS.md` para transformá-lo em uma experiência premium no estilo [ESTILO]. Foque em hierarquia visual e animações cinematográficas."*

## 5. Prompt de Experiência Imersiva (Scrollytelling)
> *"Argus, crie um blueprint de experiência imersiva para [NOME DO PROJETO]. Utilize o Protocolo de Scrollytelling de `docs/SCROLLYTELLING.md`. Implemente Scroll Interlocking para os elementos de destaque, Z-index Layering para o título principal e garanta que a física de movimento siga a Regra de Inércia (ease power4.out). O visual deve parecer um aplicativo de luxo, utilizando a stack Next.js + GSAP + Lenis."*
